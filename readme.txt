== Installation ==

= Install =

1. Get "WP Open Graph Protocol" from "Plugins > New add" on the WordPress dashboard.
2. Activate this.
3. Set it on the "Settings > WP Open Graph Protocol" in the dashboard.
4. You can set TOC show/hide individually on the post/page edit screen.

= インストール =

1. 「プラグイン > 新規追加」から「WP Open Graph Protocol」を入手します。
2. プラグインを有効化します｡
3. ダッシュボードの「設定 > WP Open Graph Protocol」で初期設定をします。
4. 投稿/固定ページの編集画面で、目次の表示/非表示を個別に設定できます。
